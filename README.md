# Gradient Descent Notes

This repository contains structured and easy-to-understand notes on Gradient Descent, a key optimization algorithm used in machine learning.

## Overview

Gradient Descent is used to minimize a cost function by iteratively adjusting model parameters. It plays an important role in training machine learning models, especially in algorithms like linear regression and neural networks.

## Topics Covered

- Introduction to Gradient Descent  
- Working and intuition  
- Mathematical formula  
- Learning rate concept  
 

## Purpose

These notes are designed for beginners and students who want a clear understanding of gradient descent and a quick revision resource.
